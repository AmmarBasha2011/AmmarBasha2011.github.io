üîß Technical Documentation

The application includes a cronjob system to schedule background tasks managed via the ammar CLI.

Cron Runner Script

The core script core/cron/cron_runner.php is used to execute tasks.

/usr/bin/php /path/to/project/core/cron/cron_runner.php <TaskName>

CLI Commands

  • php ammar list:cron: Lists all available cron tasks.
  • php ammar make:cron <TaskName>: Creates a new cron task file.
  • php ammar run:cron <TaskName>: Manually runs a specific cron task.
  • php ammar delete:cron <TaskName>: Deletes a specific cron task.
  • php ammar clear:cron: Deletes all cron tasks.

A lightweight, dependency-free JavaScript motion engine for CSS animations, controlled via the .env file.

Enabling the Engine

Set USE_ANIMATE=true in your .env file.

JavaScript `animate` function

Syntax: animate(elementOrSelector, animationName, durationMs)

// Example: Animate an element with id 'myBox'
animate('#myBox', 'fade-in', 300);

Adding New Animations

  1. Define @keyframes in public/css/motion-animations.css.
  2. Create a corresponding CSS class (e.g., .motion-slide-up).
  3. Use the new animation name in your JavaScript code.
