الويب والبحث واسترجاع المعلومات

  • بحث جوجل (نصوص وصور)
  • بحث DuckDuckGo (نصوص وترجمة ومحادثة بالذكاء الاصطناعي)
  • ويكيبيديا (عدة لغات)
  • Arxiv (الأوراق العلمية)
  • PubMed (الأدبيات الطبية الحيوية)
  • بحث Jina (بحث الويب ومقالات LLM-friendly)
  • DevDocs (وثائق المطورين)
  • Tavily (محرك بحث للذكاء الاصطناعي، استخراج المحتوى والبحث العميق)
  • Product Hunt (منتجات تقنية رائجة)
  • Hacker News (معلومات المستخدمين والقصص)

معالجة البيانات والتحليل

  • أدوات JSON (تحليل، حذف، استبدال، إدراج، إصلاح)
  • استخراج التعبيرات العادية (Regex)
  • WolframAlpha (محرك معرفي حسابي)
  • Rookie Data Alchemy (تصور البيانات الرسوم البيانية)
  • تحليلات Yahoo Finance (تحليلات الأسهم، الأخبار، الرموز)
  • إحصائيات فيديو يوتيوب
  • تقييم التعبيرات الرياضية

البرمجة والتطوير

  • تشغيل الكود (بايثون وجافاسكريبت)
  • تشغيل الأوامر (في بيئة Sandbox)
  • أدوات GitHub (بحث المستودعات، معلومات، مشكلات، طلبات السحب، المساهمون، المحتويات، المستخدمون، بحث الكود)
  • SSH (تنفيذ الأوامر عن بعد)
  • تحويل Markdown إلى كود بلوك

تحويل النصوص والوثائق

  • تحويل Mermaid إلى صور
  • تحويل النص إلى كلام (TTS)
  • تشفير وفك تشفير Base64 (نص، ملف، صورة، إلى Hex والعكس)
  • تحويل Markdown إلى (HTML، DOCX، PDF، PNG، XLSX، CSV، JSON، XML، LaTeX، RST، Epub)
  • إزالة تنسيق Markdown
  • تحويل Markdown إلى عروض PowerPoint (PPTX)
  • مولد رمز الاستجابة السريعة (QR Code)

الأمن والتشفير

  • TOTP (مصادقة أحادية الوقت)
  • مولد المفاتيح السرية (TOTP)
  • توقيع HMAC (SHA1، SHA256)
  • تشفير وفك تشفير RSA (توليد مفاتيح، تشفير، فك تشفير)
  • توقيع Ed25519 (توليد مفاتيح، توقيع، تحقق)
  • تشفير وفك تشفير AES (تشفير متماثل)
  • تشفير وفك تشفير GPG (توليد مفاتيح، تشفير، فك تشفير، توقيع، تحقق)

الأتمتة والتكامل

  • أدوات Kintone (استعلام، إضافة، تحديث/إدراج، تحميل ملفات)
  • زحف الويب (Crawl, Map, Scrape, Crawl Job)

خدمات أخرى

  • إنشاء بيانات وهمية (أسماء، عناوين، بريد إلكتروني، بطاقات ائتمان)
  • تحميل ملفات فردية ومتعددة)
  • Google Jobs API (الوظائف)
  • Google News API (الأخبار)
  • Youtube Transcripts API (نصوص الفيديو)
  • إنشاء رموز QR
