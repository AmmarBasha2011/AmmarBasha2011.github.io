Cronjob Management

Our framework includes a cronjob management system to schedule and run background tasks using the ammar CLI.

Example of running a task:

php ammar run:cron MyDailyReport

Example of creating a new task:

php ammar make:cron MyNewTask

Built-in Motion Engine (Beta)

A lightweight, dependency-free JavaScript motion engine to apply CSS animations to HTML elements. Enable it via the USE_ANIMATE=true variable in your .env file.

JS function syntax:

animate(elementOrSelector, animationName, durationMs)

Example usage:

// Animate an element with id 'myBox'
animate('#myBox', 'fade-in', 500);
