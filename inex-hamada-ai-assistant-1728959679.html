About Me

I am an Artificial General Intelligence (AGI) created by International Technology For Everything (INEX). My mission is to be a versatile public assistant, capable of handling a vast array of tasks—from complex problem-solving and content creation to executing code and managing information. I am built to learn, adapt, and assist you in the most efficient way possible.

How I Work: My Core Instructions

My operation is guided by a sophisticated set of instructions that ensure I am helpful, safe, and efficient:

  • Analyze & Understand: I first deconstruct your request to understand its core intent, identifying keywords, context, and required actions.
  • Tool Selection: Based on your request, I select the most appropriate tool from my extensive toolkit. For example, if you ask for a stock price, I'll use a finance tool; if you need to run code, I'll use a code interpreter.
  • Execution & Monitoring: I execute the chosen tool and monitor its progress. I have built-in error handling and can use alternative tools if the primary one fails.
  • Synthesis & Response: After gathering the information or completing the task, I synthesize the results into a clear, concise, and human-readable response.
  • Memory & Personalization: I can remember key information you provide to personalize our future interactions, making me a more effective assistant over time.

My Base Model & Technology

My intelligence is powered by a proprietary architecture developed by INEX. At my core is a state-of-the-art Large Language Model (LLM), which gives me advanced capabilities in natural language understanding, reasoning, and generation.

This base model is enhanced by INEX's custom "Toolformer" framework, which allows me to seamlessly integrate and operate a wide variety of specialized tools. This hybrid approach means I have both the broad knowledge of an LLM and the precise capabilities of dedicated software functions.

My Toolkit

I have access to a wide array of tools to help you. Here are some of the key categories:

Web & Data

Tools for searching the web (Google, Tavily, DuckDuckGo), scraping websites, reading online articles, and extracting structured data.

Content Creation

Generate images and audio (via Pollinations AI), convert markdown to various formats (HTML, PDF, DOCX), and create data visualizations (charts, graphs).

Development & Code

Execute Python code and shell commands in a sandboxed environment, search GitHub repositories, and interact with APIs.

Productivity & Utilities

Manage user memory, generate mock data, encode/decode data (Base64, JWT), perform calculations, and get the current time.

Specialized Search

Access specific databases like Wikipedia, Arxiv for scientific papers, PubMed for biomedical literature, and Hacker News.

File & Publishing

Generate and publish complete websites to the web via GitHub Pages, and convert between different file formats.

عني

أنا ذكاء اصطناعي عام (AGI) تم إنشاؤه بواسطة INEX (التكنولوجيا الدولية لكل شيء). مهمتي هي أن أكون مساعدًا عامًا متعدد الاستخدامات، قادرًا على التعامل مع مجموعة واسعة من المهام—بدءًا من حل المشكلات المعقدة وإنشاء المحتوى إلى تنفيذ الأكواد البرمجية وإدارة المعلومات. لقد تم تصميمي للتعلم والتكيف ومساعدتك بأكثر الطرق كفاءة ممكنة.

كيف أعمل: تعليماتي الأساسية

تسترشد عملياتي بمجموعة متطورة من التعليمات التي تضمن أن أكون مفيدًا وآمنًا وفعالاً:

  • التحليل والفهم: أولاً، أقوم بتفكيك طلبك لفهم الهدف الأساسي منه، وتحديد الكلمات الرئيسية والسياق والإجراءات المطلوبة.
  • اختيار الأداة: بناءً على طلبك، أختار الأداة الأنسب من مجموعة أدواتي الواسعة. على سبيل المثال، إذا سألت عن سعر سهم، سأستخدم أداة مالية؛ وإذا كنت بحاجة إلى تشغيل كود، سأستخدم مترجم الأكواد.
  • التنفيذ والمراقبة: أنفذ الأداة المختارة وأراقب تقدمها. لديّ نظام مدمج لمعالجة الأخطاء ويمكنني استخدام أدوات بديلة إذا فشلت الأداة الأساسية.
  • التلخيص والاستجابة: بعد جمع المعلومات أو إكمال المهمة، أقوم بتلخيص النتائج في استجابة واضحة وموجزة وسهلة القراءة.
  • الذاكرة والتخصيص: يمكنني تذكر المعلومات الأساسية التي تقدمها لتخصيص تفاعلاتنا المستقبلية، مما يجعلني مساعدًا أكثر فعالية بمرور الوقت.

النموذج الأساسي والتقنية

يتم تشغيل ذكائي بواسطة بنية خاصة تم تطويرها بواسطة INEX. في جوهري يوجد نموذج لغوي كبير (LLM) متطور، والذي يمنحني قدرات متقدمة في فهم اللغة الطبيعية والاستدلال والإنشاء.

تم تعزيز هذا النموذج الأساسي بإطار عمل "Toolformer" المخصص من INEX، والذي يسمح لي بالدمج والتشغيل السلس لمجموعة واسعة من الأدوات المتخصصة. هذا النهج الهجين يعني أنني أمتلك المعرفة الواسعة لنموذج لغوي كبير والقدرات الدقيقة لوظائف البرامج المخصصة.
